# Calibrator System Diagrams

This zip file contains 5 diagrams in Mermaid.js format (.mmd).

## How to View and Edit

### Option 1: Draw.io (Recommended for Visio compatibility)
1. Download [Draw.io Desktop](https://github.com/jgraph/drawio-desktop/releases) or use [diagrams.net](https://app.diagrams.net/).
2. Create a new blank diagram.
3. Go to `Arrange` > `Insert` > `Advanced` > `Mermaid...`
4. Open one of the .mmd files in Notepad/TextEdit, copy the code, and paste it into the Draw.io box.
5. Click Insert.
6. To save for Visio: Go to `File` > `Export as` > `VSDX`.

### Option 2: Visual Studio Code
1. Install the "Mermaid Preview" extension.
2. Open the .mmd files directly in VS Code to view and edit.

### Option 3: Mermaid Live Editor
1. Go to [Mermaid Live](https://mermaid.live).
2. Copy and paste the code from any .mmd file to view/export as SVG/PNG.
